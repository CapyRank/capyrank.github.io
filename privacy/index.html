Privacy Policy

Last updated: March 2026

Overview

CapyRank is a community-voted streamer discovery platform. This policy explains what data we collect, how we use it, and how we protect it. We keep things simple — we collect only what's necessary to run the platform and nothing more.

Data We Collect

When a streamer joins CapyRank via Twitch OAuth, we store the following information in our database:

  • Twitch user ID and username
  • Display name and profile picture URL
  • Follower count (updated periodically)
  • Live status (updated periodically)
  • Capy counts (given and received)

When a viewer gives Capys in chat, we store their Twitch user ID, username, display name, profile picture URL, and a record of how many Capys they've given and to whom.

We do not collect email addresses, passwords, payment information, or any sensitive personal data.

How We Use Your Data

Data collected is used solely to operate CapyRank:

  • Displaying streamer profiles and leaderboards on capyrank.com
  • Tracking Capy allocations and supporter rankings
  • Powering the CapyRank Twitch Extension panel
  • Showing viewers how many Capys they have remaining each day

We do not sell, share, or use your data for advertising, analytics, or any purpose other than operating the platform.

Twitch Extension

The CapyRank Leaderboard Twitch Extension displays the top supporters for a channel. It reads publicly available CapyRank data from our database. The extension does not collect any personal information from viewers beyond what is already stored as described above. No identity linking or viewer authentication is used.

Data Storage

Data is stored in Google Firestore (Firebase), hosted in the United States. Google's privacy practices apply to data stored there. You can review Google's privacy policy at policies.google.com/privacy.

Data Retention & Deletion

Capy counts reset on a daily, weekly, monthly, and yearly basis as part of normal platform operation. All-time totals are retained indefinitely while a streamer remains registered.

If you'd like your data removed from CapyRank, contact us at hello@capyrank.com and we'll delete your records promptly.

Third-Party Services

CapyRank uses the following third-party services:

  • Twitch API — for authentication and user data
  • Google Firestore — for data storage
  • Cloudflare — for DNS, SSL, and the join flow
  • GitHub Pages — for hosting the website

Contact

If you have any questions about this privacy policy or how your data is handled, contact us at hello@capyrank.com.
